Instaling Helm on OS X

Installing Helm (0.6.1 at the time of writing) on Mac OS X turned out to be a bit of a chore. I don't have Haskell-platform, I have [Haskell for Mac OS X](http://ghcformacosx.github.io). Here's what I did to get Helm installed:

$ cabal sandbox init
$ brew install sdl2
$ brew install cairo --without-x
$ brew install pango --without-x
$ cabal install gtk2hs-buildtools
$ env PATH=$(pwd)/.cabal/bin:$PATH PKG_CONFIG_PATH=/opt/X11/lib/pkgconfig cabal install "helm>=0.6.1" --with-gcc=gcc-4.9

The PATH setting is required so that the gtk2hs-buildtools binaries are found, and PKG_CONFIG_PATH so that the pkg-config files for the homebrew-installed libraries are found. Finally, it's necessary to give the --with-gcc flag or glib compilation fails.
